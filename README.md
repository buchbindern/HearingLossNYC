# HearingLossNYC
This is my semester project for my Linux Programing & Data Analysis course. We had to make a hypothesis and use data to prove or disprove it. 
Hypothesis: NYC residents have higher rates of hearing loss than those in greater NYS.
I used zipcode, census, and NPI data.
I compare the ear doctors per capita in every NYC and NYS zipcode, percent of people over 65 in each zip and visualize my findings. 

## Features
- Data processing using AWK scripts and Python.
- Visualizations created with Matplotlib.

## Additional Data Needed
This project requires the following datasets:

1. **npi.gz**: [Download here](http://download.cms.gov/nppes/NPI_Files.html)

2. **uszips.csv**: [Download here](https://simplemaps.com/data/us-zips)
